  • ViewControllerBussinessService

    Another complicated function.

    Remark

    There’s a counterpart function that concatenates the first and last name into a full name.

    See also

    createFullName(_:lastname:)

    Precondition

    fullname should not be nil.

    Requires

    Both first and last name should be parts of the full name, separated with a space character.

    Todo

    Support middle name in the next version.

    Warning

    A wonderful crash will be the result of a nil argument.

    Version

    1.1

    Author

    Myself Only

    Note

    Too much documentation for such a small function.

    Usage Example:

    let single = 5
let double = doubleValue(single)
print(double)
    • Use the doubleValue(_:) function to get the double value of any number.
    • Only Int properties are allowed.
